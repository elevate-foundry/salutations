Getting Started

Install Autonomous Git and start letting AI handle your version control in minutes.

Installation

Clone the Salutations repository and install dependencies:

git clone https://github.com/elevate-foundry/salutations.git
cd salutations
pip install -r requirements.txt

Quick Start

Try the autonomous Git agent in your repository:

from pathlib import Path
from examples.autonomous_git_agent import AutonomousGitAgent

# Initialize agent
agent = AutonomousGitAgent(Path.cwd())

# Let agent decide if you should commit
agent.auto_commit()

# Agent will:
# ‚úì Analyze your changes
# ‚úì Calculate fitness score
# ‚úì Generate commit message
# ‚úì Commit if fitness is high enough

Basic Usage

Auto Commit

Let the agent decide when to commit based on fitness:

agent.auto_commit()  # Commits if fitness > 0.5

Force Commit

Commit regardless of fitness score:

agent.auto_commit(force=True)

Smart Branching

Let agent suggest and create branches:

# Agent suggests branch strategy
strategy = agent.suggest_branch_strategy("fix login bug")
# Returns: fix/fix-login-bug with squash merge strategy

# Auto-create and switch to branch
branch = agent.auto_branch("add user authentication")
# Creates: feature/add-user-authentication

Auto Merge

Merge with intelligent strategy selection:

agent.auto_merge("feature/new-feature", "main")
# Agent chooses best merge strategy
# Resolves conflicts automatically

Running as Daemon

Install as a background service that runs 24/7:

Install Service

python examples/git_daemon.py --install --repo /path/to/repo

Configuration Options

# Check every 5 minutes (default)
--interval 300

# Auto-commit threshold (0-1)
--threshold 0.7

# Real-time mode (1 second interval)
--realtime

Control Commands

# Start daemon
launchctl start com.salutations.git-daemon

# Stop daemon
launchctl stop com.salutations.git-daemon

# View logs
tail -f .git/daemon.log

# Uninstall
python examples/git_daemon.py --uninstall

Fitness Functions

The agent uses a fitness function to determine commit quality:

üìÅ File Count (30%)

Sweet spot: 3-7 files. Too few or too many reduces fitness.

üéØ Cohesion (20%)

Related files (same directory/type) increase fitness.

üß™ Tests (20%)

Including test files significantly boosts fitness.

üìù Documentation (15%)

Updated docs show thoroughness.

üìç Localization (15%)

Changes in few directories indicate focus.
